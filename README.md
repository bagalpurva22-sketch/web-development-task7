# Laundry Service Landing Page

## Overview
This is a responsive Laundry Service Landing Page built using HTML and CSS. The website provides information about laundry services and allows users to book a service through a simple and attractive interface.

## Features
- Responsive Design
- Navigation Bar
- Hero Section
- Service Description
- Call-to-Action Button
- Laundry Image
- Mobile and Tablet Friendly Layout

## Technologies Used
- HTML5
- CSS3

## Files
- index.html
- style.css
- laundry1.png

## How to Run
1. Download the project files.
2. Open the project folder in VS Code.
3. Open `index.html` using Live Server.
4. View the website in your browser.
